# Q2 – Profile Header + Overlay Card

## Overview
A profile screen header using Box layering in Jetpack Compose. The page includes a gradient background, circular-shaped avatar photo, and overlay Card stack together to create the profile card look where the avatar overlaps the header and card. Demonstrated the profile page with a variety of Material 3 components and modifiers.

## Material 3 Components Used
Card, Icon, Button, FilledTonalButton, AssistChip, TopAppBar

## Modifiers Demonstrated 
• clip(CircleShape) for avatar
• offset 
• zIndex 
• shadow 
• elevation via Card defaults
• fixed size

## AI Disclosure
Assist Chip design and components that make up the profile page's look, such as "posts", "followers", and "share", were assisted by AI.
